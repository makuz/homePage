Currently my fields of interest are focused on how can I combine my music skills and my programming skills.
To develop this interests I am using now among others the Sonic PI environment.
Sonic PI is something like a programming language environment for creating music.
As I go further into programming I begin to see more and more similarities between programming and composing music.
This similarities I see for example in comparing design patterns and music forms or list of objects comparing to scales in music.
As for my current interest in programming I am excited now about big data and machine learning concepts.
